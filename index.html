Entities
UserAccount
Player account record. PK = user_id, SK = "header"
GuestAccount
Anonymous account lifecycle. PK = user_id, SK = "guest"
LinkedPlatform
PC platform link (Steam). SK = platform#header#<PLATFORM>#uid#<id>. UID is globally unique.
LinkedConsole
Console link (PlayStation, Xbox). SK = console#header#<CONSOLE>#uid#<id>. UID is globally unique.
LinkedSocial
Network identity (Social). SK = app#<app_key>#social#<provider>#uid#<id>. Used for social login via network (e.g. "google"). Scoped by app — social UIDs aren't globally unique.
LinkedProfile
Network identity (OTP). SK = profile#provider#<provider>#id#<id>. Used for email OTP via network "auth-provider".
Key pattern
Single-table design. PK is always the user_id (usr_…). SK encodes the record type and qualifiers.
